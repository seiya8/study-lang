## 3.5 fmtパッケージ

- `println()`, `print()`は標準エラー出力。`fmt.Println()`や`fmt.Printf()`は標準出力。
- `fmt.Printf()`の使い方。
  - `%d`: 10進数, `%b`: 2進数, `%x`: 16進数, `%s`: 文字列
  - `%v`はどんな型でもOK。配列など。

## 3.6 変数

- 値型、参照型（スライス、マップ、チャネル）、ポインタ型の3種類。
- パッケージ変数とローカル変数。
- 定義したが参照されない変数があるとコンパイルエラー。
- 明示的な定義をして代入
  ```go
  var n int
  var x, y, z int
  var (
    i, j int
    name string
  )
  i, j = 1, 2
  ```
- 暗黙的な定義（型推論）
  ```go
  i := 1 // int
  f := 3.14 // float64
  i = 2 // これはOKだが、
  i := 2 // 再定義は無理
  ```
- 複数変数の暗黙的な定義
  ```go
  var (
    n = 1
    s = "string"
  )
  ```
  これは、`n := 1`と`s := "string"`を並べるより見やすいため好まれる。
